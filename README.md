## Usage

### Viewing Designs
- **Images**: Open `.jpg` and `.png` files directly in any image viewer
- **Videos**: Play `.mp4` files using any video player or social media platform
- **Folders**: Each subfolder contains related design assets

### For Designers
- Use these designs as inspiration for your own projects
- Modify and adapt for personal or commercial use (check licensing)
- Study color schemes, typography, and composition techniques

### For Events
- Download and use for promotional materials
- Share on social media platforms
- Print for physical displays
